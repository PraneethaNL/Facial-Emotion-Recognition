# Emotion recognition with CNN

Implemented a CNN model to correctly classify facial expressions corresponding to the basic emotions.

## Dependencies

- [NumPy](http://docs.scipy.org/doc/numpy-1.10.1/user/install.html)
- [Tensorflow](https://www.tensorflow.org/versions/r0.8/get_started/os_setup.html)
- [TFLearn](https://github.com/tflearn/tflearn#installation)
- [OpenCV](https://opencv-python-tutroals.readthedocs.io/en/latest/)

## Dataset 
[FER-2013 Faces Database]

## Usage

$ python emotion_recognition.py poc
